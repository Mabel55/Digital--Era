Django is a high-level Python web framework that follow a 'batteries-included' philosophy, meaning it comes pre-packaged with built-in toools for user authenciation, adminstrative dashboards, database management, and security out of the box. Django follows the Model-View-Template(MVT) architecture. The Model handles your database structure, the template handles the user interface layout, and the view acts as the bridge containing the core buisness logic that process requests and returns responses.
