Flask is a micro-framework, meaning it provides the bare essentials for web development (routing, request handling, and templating) without forcing a specific database tool or project structure on the developer. This gives you absolute architectural freedom. Routing in Flask is handled using Python decorators. For example, decorating a function with '@app.route("/home")' maps that specific URL directly to a Python function that executes and returns an HTTP response, HTML template, or a JSON payload to the client.
