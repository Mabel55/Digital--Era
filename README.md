# Mabel Academy 🎓🚀

Mabel Academy is a high-performance, cloud-deployed EdTech platform built to bridge the gap between foundational programming and advanced AI Engineering. The platform features a fully integrated, browser-based execution environment and a contextual AI instructor to guide students through real-world software development tracks.

Live Platform: [digital-era.live](https://digital-era.live)

---

## 🛠️ Tech Stack & Architecture

- **Backend Framework:** FastAPI (Asynchronous, high-performance Python framework)
- **Database:** PostgreSQL / SQL Relational Database Management
- **Containerization & Deployment:** Docker & Docker Hub
- **Cloud Infrastructure:** Azure App Services (with Continuous Integration webhooks)
- **AI Ecosystem:** Large Language Models (LLMs), LangChain, and Retrieval-Augmented Generation (RAG)

---

## ✨ Core Features

### 🖥️ 1. Integrated Browser Code Editor
Students can write, execute, and test Python code natively inside their browser dashboard. This removes the friction of local environment setups for beginners and ensures immediate, interactive feedback loops.

### 🤖 2. Interactive AI Instructor
Integrated via advanced LLM prompting and RAG pipelines, the built-in AI tutor acts as a virtual senior developer:
- Provides automated code grading against lesson constraints.
- Delivers real-time error analysis and structural suggestions.
- Uses contextual course data to answer student queries dynamically.

### 📜 3. Automated Certificate Generation
A seamless custom backend pipeline triggers automated PDF certificate generation immediately upon a user's successful completion of any structured learning track.

---

## 📚 Learning Tracks

1. **Python Core:** From variables and data structures to full Object-Oriented Programming (OOP).
2. **SQL & Relational Databases:** Schema design, querying, joins, and indexing optimization.
3. **Backend Engineering:** Server-side application development, routing, security, and API creation using **FastAPI**.
4. **Data Science & AI Engineering:** Practical implementation of Neural Machine Translation, LLMs, and RAG architectures.
